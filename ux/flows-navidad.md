# UX Flow: Navidad (Christmas)

## 1. Landing Page (Home)
- **Hero**: "Transforma tus fotos en recuerdos navideños mágicos".
- **CTA**: "Crear mi foto ahora".
- **Examples**: Carousel of before/after images.
- **Packages**: Display 3 main packages (Single, Couple/Pet, Family).

## 2. Upload & Configuration
- **Upload**: User uploads 1-3 photos.
- **Configuration**:
    - Select number of people (Adults, Children).
    - Select number of pets (Dogs, Cats, Others).
    - Select "Vibe" (Cozy, Elegant, Fun, Snowy).
- **AI Help**: Optional prompt enhancement.

## 3. Checkout
- **Summary**: Show selected package and configuration.
- **Pricing**: Dynamic price calculation.
- **Payment**: Stripe/MercadoPago integration.
- **Referral**: Input referral code for discount/credit.

## 4. Processing
- **Status**: "Generating your magic..." with progress bar.
- **Wait Time**: Estimated time (e.g., "Ready in ~5 mins").

## 5. Result & Download
- **Gallery**: View generated images.
- **Actions**: Download High-Res, Share, Generate More.
- **Referral**: "Share your code to get free photos!".
